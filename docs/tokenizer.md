# Tokenizer Generation

The tokenizer files are generated by CMake into the active build directory:

- `generated/tokenizer_map.cc`
- `generated/tokenizer_map.hpp`
- `generated/token_map.json`

The source tree should not contain a checked-in `token_map.json`. Rebuild the
`generate_tokenizer_map` target to refresh all generated tokenizer artifacts for
the selected build directory.
